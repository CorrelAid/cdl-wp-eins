# 

Dieses Material bietet Dokumentation und Tools zu verschiedenen Entrypoints in den Lebenszyklus eine Umfrage

- Von Anfang an: 
    - Design von Umfragen mit xlsforms
        - Einführung in Antworttypen/Logik (Mapping von xlsforms typen auf Theorie aus Umfrageforschung)
    - Umwandlung in kompatible Formate für den Import in verschiedene Tools (js libraries und webapp)
    - Datengenerierung
        - Python code für die Generierung von Testdaten mit LLMs
    - Auswertung
        - Dokumentation für Auswertung in den Tools 
        - Umwandlung von Exporten aus den Tools in standardisierte Form (DDI Codebook) (python package)
        - Weiterführende statistische Auswertung

- Bestehende Survey in verschiedenen Tools
    - Umwandlung von Exporten aus den Tools in standardisierte Form (DDI Codebook) (python package)

In diesem Material wird der XLSForm-Standard (und ODK XForms) vorgestellt und wie man ihn nutzen kann, um Best Practices im Survey Design und der Auswertung einzuhalten. 

Außerdem wird ein Tool vorgestellt, mit dem man Fragebögen, die diesem Standard entsprechen, in verschiedene Survey-Anwendungen importieren kann:

-	LamaPoll
-	SoSci Survey
-	LimeSurvey

KoboToolbox unterstützt XLSForm bereits.

Wir geben  konzeptionelle Einleitungen in die Antworttypen des XLSForm-Standards, erläutern, wann man welchen Typ verwenden sollte mappen darauf Auswertungs- und Visualisierungsoptionen.

Das Material ist aufgeteilt in die Abschnitte **Umfragedesign** und **Umfrageauswertung**. In Umfragedesign wird in den XLSForm-Standard eingeführt. In **Umfragedesign**
